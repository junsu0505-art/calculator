# 계산기 (Calculator)

순수 HTML/CSS/JS 단일 파일 계산기 웹앱입니다. Google Material Design 테마를 기반으로 제작되었습니다.

## 기능

- 사칙연산 (+, -, ×, ÷)
- 퍼센트 (%)
- 부호 변경 (+/−)
- 백스페이스 (⌫)
- 메모리 4종: MC · MR · M+ · M−
- 계산 기록
- 키보드 입력 지원
- 반응형 레이아웃
- 0으로 나누기 에러 처리

## 실행 방법

별도 설치 없이 브라우저에서 `index.html` 파일을 열면 바로 실행됩니다.

```
index.html 파일을 더블클릭하거나 브라우저에 드래그&드롭하세요.
```
